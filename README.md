# 🌿 **環保小尖兵 (Eco-Guardian)**

這是一個專為國小中低年級學生設計的網頁互動遊戲，旨在透過「情境化命題」與「行為後果」的即時回饋，將抽象的環保知識轉化為具體的日常抉擇，培養學生的環境保護意識。

## 📖 遊戲設計理念
在「環保小尖兵」的世界中，玩家將化身為守護地球的行動者。本遊戲核心在於**「連結行為與影響」**。

我們觀察到傳統教學往往只給予「正確/錯誤」的斷定，學生難以連結生活與環境的關係。因此，本遊戲將每道題目轉化為一個**真實情境**，讓學生在選擇後直接看見該行為產生的「正面結果」或「負面後果」，進而引發深度省思。

## 🌟 核心特色

### 1. **明確的情境化導引**
- **標題主題化**：每題設有具體標題（如：🚰 刷牙省水、📦 網購包裝處理），讓學生一眼看出環境主題。
- **敘述完整化**：捨棄簡略的詞彙，採用「情境 + 動作引導」的問法（例如：「當你喝完鮮奶後，關於包裝的處理，你會怎麼做？」），確保教學意義精準傳達。

### 2. **行為後果回饋系統**
- **動態回饋**：根據學生的選擇，即時呈現具體的後果描述（如：省下大量乾淨水資源 vs. 造成海洋生物受害）。
- **邏輯強化**：透過因果關係的呈現，訓練學生的邏輯思考與價值判斷。

### 3. **海量隨機題庫**
- **80 題全方位情境**：涵蓋水資源、廢棄物管理、減碳飲食、海洋友善、公共節能及生物多樣性等六大領域。
- **動態隨機機制**：每次啟動僅抽取 5 題任務，確保學生每次遊玩的體驗與挑戰皆不相同。

### 4. **教育導向的結算頁面**
- **積分稱號系統**：根據表現給予「偉大的環境守護者」至「環境路人甲」等趣味稱號。
- **行為省思紀錄**：完整列出本次挑戰的決策清單，方便教師或家長在課堂、家庭中進行後續的引導討論與「混合式學習」。

## 🛠️ 技術開發
- **語言架構**：採用純 HTML5 / CSS3 / JavaScript 開發。
- **輕量化設計**：無須安裝、無須外掛，完美支援電腦、平板及手機等跨平台裝置。
