# 🌿 **環保小尖兵 (Eco-Guardian)**

這是一個專為國小學生設計的網頁互動遊戲，旨在透過「行為與後果」的即時回饋，培養學生的環境保護意識與社會責任感。


## 🎮 立即體驗
[點擊這裡開始遊戲](https://weddy1225-blip.github.io/AI-social-study04-v2-/)

---

## 📖 遊戲背景
在「環保小尖兵」的世界中，玩家的每一個選擇都至關重要。這不是一場單純的對錯考試，而是一次模擬真實生活的冒險。當你選擇省水，你會看到村莊的繁榮；當你選擇浪費，你將親眼見證環境的惡化。

## 🌟 特色
- **行為後果機制**：打破傳統「正確/錯誤」的框架，根據學生的選擇給予具體的「正面結果」或「負面後果」文字回饋，強化邏輯思考。

- **全方位題庫**：內建 47 題 涵蓋水資源、廢棄物處理、節能減碳、自然生態及公共秩序等主題。

- **動態難度**：每次遊戲隨機抽取 5 題任務，確保每次遊玩的體驗都有所不同。

- **積分視覺化**：最終結算頁面提供「環保積分」與詳細的「行為紀錄清單」，方便老師或家長進行後續引導討論。

- **輕量化設計**：採用純 HTML5 / CSS3 / JavaScript 開發，無須安裝外掛，支援跨平台瀏覽。
